This tool WBBMTT is yet another multi-touch screen test program implemented for HTML5 Web browser. Android application YAMTT (Yet Another MultiTouch Test, Google Play Store, dev blog ) influences for basic functionality and graphic design of WBBMTT. Thanks for good tool.
